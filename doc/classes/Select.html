Parameters:

  • [options] Object optional

    Configuration object

    • [checkedAllLabel] String optional

      Label to display if all items of select are checked

    • [data] Array optional

      array of data [string, object]

    • [defaultLabel] Array optional

      default label which gets displayed

    • [deselectCallback] Function optional

      callbackfunction, when element is deselected

    • [deselectField=false] String | Boolean optional

      allows deselection in case of single select; will be set if value is a string

    • [disabled=false] Boolean optional

      disables the select field if set to true

    • [instanceName] String optional

      instance name of this component

    • [multipleSelect=false] Boolean optional

      allows multiple elements to be selected

    • [preSelectedElements] Array optional

      allows preselection of fields by defining the id attributes or strings

    • [selectCallback] Function optional

      callbackfunction, when element is selected

    • [valueName] String optional

      name of property which should be used

    • [style] String optional

      "normal", "small", "big", "action" for different appearance

    • [skin] String optional

      style class to set to the elements component (e.g 'white')

    • [emitValues] Boolean optional

      If true the value is emited with events instead of the id

    • [fixedLabel] Boolean optional

      If true the label never gets changed

    • [icon] String optional

      icon to set into the label

    • [noItemsCallback] Function optional

      callback function which gets executed at click on label if no dropdown-items exist

    • [repeatSelect] Boolean optional

      If true dropdown item can be selected n-times in succession
