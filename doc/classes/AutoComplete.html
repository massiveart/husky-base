    • [stickToInput] Boolean optional

      If true suggestions are always under the input field

    • [hint] Boolean optional

      if false typeahead hint-field will be removed

    • [emptyOnBlur] Boolean optional

      If true input field value gets deleted on blur

    • [excludes] Array optional

      Array of suggestions to exclude from the suggestion dropdown
